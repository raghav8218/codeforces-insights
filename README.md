**Codeforces Insights** is a React-based web app that visualizes a user's Codeforces submissions. It helps analyze problem-solving patterns by grouping submissions by verdicts like Correct, Wrong Answer, Time Limit Exceeded, and Others. You can also sort them based on difficulty rating.

## ğŸš€ Features
- **Enter your Codeforces handle** and fetch your recent submissions.
- **Grouped verdicts:**
  - Correct
  - Wrong Answer (only first attempt per problem)
  - Time Limit Exceeded
  - Other verdicts (Compilation Error, Skipped, etc.)
- **Sort problems** by difficulty rating (ascending/descending)
- **Submission timestamps** for each problem.
- **Clickable links** to each problem on Codeforces.
